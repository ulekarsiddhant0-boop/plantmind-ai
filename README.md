# PlantMind AI

AI-Powered Industrial Knowledge Intelligence Platform

## Problem

Industrial knowledge is scattered across manuals, SOPs, maintenance reports, and incident reports. Engineers spend significant time searching for information.

## Solution

PlantMind AI enables users to:

- Upload industrial PDFs
- Generate document summaries
- Ask questions from documents
- Analyze incidents
- Receive AI-powered recommendations

## Features

- PDF Upload
- Document Text Extraction
- AI Summary Generation
- Question Answering
- Incident Analysis

## Technology Stack

- Python
- Streamlit
- PyPDF
- Google Gemini API

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
